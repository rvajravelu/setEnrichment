To install setEnrichment package for R:

1. Download: setEnrichment_2.0.0.tar.gz
2. Install package to R using instructions here: https://r-coder.com/install-r-packages/#Installing_packages_in_R_from_zip_source
3. Attached package using command: library(setEnrichment)
4. For operating instructions, use command: ? setEnrichment
5. Sample databases included in GitHub directory: "smallFakePWAS.csv", "smallFakePWAS_withClassLabels.csv", and "cdiPWAS_60dayExposure.csv"
